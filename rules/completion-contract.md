# Completion Contract

> Extracted from CLAUDE.md for noyeah-harness

NO completion claims without ALL of:

1. Fresh test output (pass/fail count, run NOW not before)
2. Fresh build output (exit code 0)
3. Zero pending TODO items
4. Architect verification (STANDARD tier minimum, THOROUGH for >20 files or security)
5. State file updated to terminal phase

Words that CANNOT appear in completion claims: "should", "probably", "seems", "looks like"
